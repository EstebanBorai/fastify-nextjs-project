## Install dependencies

```sh
yarn install
```

### Run development server

```sh
yarn dev
```

### Create bundle for production

```sh
yarn build
```

### Serve production bundle

```sh
# This command requires `yarn build`
yarn start
```

### Lint project files

```sh
yarn fmt
```
